# DashboardGraphQL

Example of using graphql to return datas to frontend.

## Deploying

1. Start server
2. Create file .env
3. Add value ENGINE_API_KEY=service:<serviceid>:<key>
4. Execute the command below

```
npm run start
npx apollo service:push --endpoint=http://localhost:4000
```
